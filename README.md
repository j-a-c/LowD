LowD
=====================
LowD is an infinite exploration game.

## Run
All binaries can be found in the folder *bin*.

### Linux
    ./bin/lowd

### Windows
    bin/lowd.exe

### Mac OS X
Not supported yet.

## Dependencies
- GLEW 1.6.0
- GLFW 3.0.4

## Compilation

### All platforms
    git clone https://github.com/j-a-c/LowD.git
    make

### Linux
    git clone https://github.com/j-a-c/LowD.git
    make linux

### Windows
    git clone https://github.com/j-a-c/LowD.git
    make windows

### Mac OS X
Not supported yet.
